# Achievements & Creator Profiles Design

Система ачивок и отображение профиля создателя в карточках пати.

## Ачивки (11 штук)

### По количеству созданных пати
| Тип | Иконка (Lucide) | Название | Условие |
|-----|-----------------|----------|---------|
| `first_party` | `PartyPopper` | Первая пати | 1 пати |
| `activist` | `Flame` | Активист | 10 пати |
| `veteran` | `Star` | Ветеран | 50 пати |
| `legend` | `Crown` | Легенда | 100 пати |

### По джоинам
| Тип | Иконка (Lucide) | Название | Условие |
|-----|-----------------|----------|---------|
| `teammate` | `Handshake` | Тиммейт | 10 джоинов |
| `team_player` | `Gamepad2` | Командный игрок | 50 джоинов |

### По стрикам
| Тип | Иконка (Lucide) | Название | Условие |
|-----|-----------------|----------|---------|
| `week_streak` | `Calendar` | Неделя огня | 7 дней подряд |
| `month_streak` | `CalendarCheck` | Месяц огня | 30 дней подряд |

### Особые
| Тип | Иконка (Lucide) | Название | Условие |
|-----|-----------------|----------|---------|
| `night_gamer` | `Moon` | Ночной геймер | пати после 00:00 |
| `quick_fill` | `Zap` | Быстрый старт | заполнилась за 5 мин |
| `sniper` | `Target` | Снайпер | 100% fill на 10+ пати |

## Отображение в карточке пати

Строка с инфой о создателе:
- Уровень: `Lv.5` (цвет: 1-5 серый, 6-10 синий, 11+ золотой)
- Количество пати: `23 пати`
- Главная ачивка: самая редкая из разблокированных

Если нет ачивок — только `Lv.1 • 0 пати`
Если аноним — строка не показывается

## Tooltip (мини-профиль)

При наведении на строку создателя:
- Аватар + никнейм
- Уровень + XP
- Статистика: пати созданы, джоины, стрик
- Ачивки: разблокированные цветные, заблокированные серые

Поведение:
- Hover с задержкой 300ms
- На мобилке — по клику

## Backend

### Новый эндпоинт
```
GET /api/users/{user_id}/profile
```

### Автоматическая выдача ачивок
- После создания пати → first_party, activist, veteran, legend, night_gamer
- После джоина → teammate, team_player
- После обновления стрика → week_streak, month_streak
- При заполнении пати → quick_fill, sniper

## Frontend компоненты

- `CreatorBadge.tsx` — строка в карточке
- `CreatorTooltip.tsx` — мини-профиль при hover
- `achievements.ts` — константы ачивок
- `useCreatorProfile(userId)` — хук для загрузки профиля
