# Communities Landing Page Design

Страница для валидации гипотезы: готовы ли админы Discord/Telegram сообществ платить за интеграцию сервиса поиска пати.

## URL

`/communities`

## Структура страницы

### Hero секция

```
Помогите вашему сообществу находить тиммейтов быстрее

Превратите активность сообщества в игровые сессии —
автоматически, без хаоса в чате.

[Попробовать бесплатно]
```

### Блок "Что вы получаете"

- ✅ Создание пати прямо из Discord / Telegram
- ✅ Автоудаление старых пати (без спама и мёртвых постов)
- ✅ Единый пул пати (ваш сервер + публичное обнаружение)
- ✅ Присоединение в один клик по ссылке
- ✅ Базовая аналитика (просмотры / присоединения)

### Тарифы

| Free | Pro — $15/мес |
|------|---------------|
| До 3 активных пати | Безлимит пати |
| Публичный листинг | Приоритет в выдаче |
| Стандартные ссылки | Брендинг сообщества |
| | Аналитика (просмотры/джоины) |
| | Кастомные инвайт-сообщения |

### Форма заявки (inline после CTA)

**Поля:**
1. Email * — текст
2. Платформа * — чекбоксы: Discord / Telegram / Оба
3. Размер сообщества * — селект: до 100 / 100-500 / 500-1000 / 1000+
4. Готовы платить $15/мес за Pro? * — радио: Да / Возможно / Нет
5. Название сообщества — текст (опционально)
6. Какие игры играете — текст (опционально)

**После отправки:** "Спасибо! Мы свяжемся с вами в ближайшее время."

## Entry points на главной

### Header
Ссылка "Для сообществ" — между логотипом и кнопками справа

### Баннер перед CTA секцией
```
Управляете Discord или Telegram сообществом?
Мы делаем инструменты для быстрого поиска пати
[Узнать больше →]
```

## Umami аналитика

```typescript
communitiesPageView()              // зашёл на /communities
communitiesCtaClick()              // клик на "Попробовать бесплатно"
communitiesFormStart()             // начал заполнять форму
communitiesFormSubmit(data)        // отправил форму (platform, community_size, willing_to_pay)
communitiesFormError(field)        // ошибка валидации
communitiesLinkClickHeader()       // клик из хедера
communitiesLinkClickBanner()       // клик из баннера
```

## Бэкенд

### Эндпоинт
```
POST /api/community-leads
```

### Таблица `community_leads`
- id (uuid)
- email (text, not null)
- platform (text) — discord/telegram/both
- community_size (text) — under_100/100_500/500_1000/over_1000
- willing_to_pay (text) — yes/maybe/no
- community_name (text, nullable)
- games (text, nullable)
- created_at (timestamp)
