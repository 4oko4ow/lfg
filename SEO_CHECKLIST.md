## 🔧 Что нужно сделать в Yandex Webmaster:

1. **Проверка сайта:**
   - Убедиться, что сайт добавлен: https://webmaster.yandex.ru/
   - Проверить статус индексации

2. **Отправить Sitemap:**
   - URL: `https://findparty.online/sitemap.xml`
   - Раздел: Индексирование → Файлы Sitemap
   - Добавить sitemap и проверить статус

3. **Проверить robots.txt:**
   - URL: `https://findparty.online/robots.txt`
   - Убедиться, что он доступен и правильно настроен

4. **Настроить переиндексацию:**
   - Раздел: Индексирование → Переобход страниц
   - Запросить переиндексацию главной страницы и важных страниц игр

5. **Проверить мобильную версию:**
   - Раздел: Индексирование → Мобильные страницы
   - Убедиться, что сайт корректно отображается на мобильных

6. **Настроить регион:**
   - Раздел: Настройки → Регион сайта
   - Установить регион (Россия/СНГ)

7. **Проверить безопасность:**
   - Раздел: Безопасность
   - Убедиться, что нет предупреждений

## 🔧 Что нужно сделать в Google Search Console:

1. **Добавить сайт:**
   - URL: https://search.google.com/search-console
   - Добавить свойство: `https://findparty.online`
   - **ВАЖНО:** Нужно добавить meta tag для Google verification
   - После добавления получить код и добавить в `index.html`:
     ```html
     <meta name="google-site-verification" content="ВАШ_КОД_ВЕРИФИКАЦИИ" />
     ```

2. **Отправить Sitemap:**
   - Раздел: Sitemaps
   - URL: `https://findparty.online/sitemap.xml`
   - Отправить и проверить статус

3. **Проверить покрытие индексации:**
   - Раздел: Покрытие
   - Проверить, какие страницы проиндексированы
   - Исправить ошибки, если есть

4. **Проверить мобильную удобность:**
   - Раздел: Удобство для мобильных устройств
   - Убедиться, что нет проблем

5. **Проверить Core Web Vitals:**
   - Раздел: Производительность
   - Проверить метрики производительности

6. **Настроить интернационализацию:**
   - Раздел: Интернационализация
   - Проверить hreflang теги

7. **Проверить безопасность:**
   - Раздел: Безопасность и нарушения
   - Убедиться, что нет проблем

## 📋 Дополнительные рекомендации:

### Для улучшения ранжирования:

1. **Регулярно обновлять контент:**
   - Обновлять sitemap.xml при добавлении новых игр
   - Обновлять lastmod даты в sitemap

2. **Мониторинг:**
   - Регулярно проверять позиции по ключевым запросам
   - Отслеживать трафик из поисковых систем
   - Анализировать CTR в Search Console

3. **Внешние ссылки:**
   - Получать обратные ссылки с игровых форумов
   - Упоминания в социальных сетях
   - Партнерства с игровыми сообществами

4. **Техническая оптимизация:**
   - Проверить скорость загрузки страниц
   - Оптимизировать изображения
   - Минифицировать CSS/JS

5. **Контент:**
   - Добавить больше статических страниц для топовых игр
   - Создать FAQ страницу
   - Добавить блог с полезным контентом
